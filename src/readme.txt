1. ¿Cómo entiende ChatGPT los promts?

ChatGPT entiende los promts mediante su algoritmo de aprendizaje. Este algoritmo se encarga de procesar
grandes cantidades de datos y generar una respuesta acorde a ellos.
El proceso por el cual ChatGPT entiende un promt implica analizar el texto proporcionado por el usuario,
identificar las palabras clave del mismo, interpretar el significado y luego generar una respuesta 
que satisfaga las necesidades del usuario

2. Dentro del algoritmo de GPT para predecir palabra tras palabra, ¿cómo se realiza este procedimiento?

El algoritmo de GPT predice mediante el siguiente procedimiento: 
Comienza tokenizando el texto en unidades discretas llamadas tokens. 
Luego, cada token se convierte en un vector numérico que representa su significado y contexto. 
Se incorpora información sobre la posición relativa de cada token en la secuencia. 
Se calcula la atención entre todos los tokens para capturar las relaciones contextuales. 
Después, los tokens pasan por capas de transformación para procesar su contexto. 
Se genera una distribución de probabilidad sobre el vocabulario para predecir la siguiente palabra. 
Finalmente, se selecciona la siguiente palabra utilizando técnicas como decodificación o muestreo 
de acuerdo a la distribución de probabilidad. 

3. ¿Cómo emite respuestas ChatGPT?

Cuando se le presenta una entrada de texto, ChatGPT procesa esa entrada utilizando su conocimiento 
y entendimiento del lenguaje. Luego, genera una respuesta que considera adecuada y relevante 
para la entrada dada.
El proceso de generación de respuestas implica la combinación de varios componentes, incluyendo la comprensión
del contexto, la selección de palabras y frases apropiadas y la generación de texto. 
Estas respuestas se generan de manera automática y en tiempo real, lo que permite a ChatGPT interactuar 
con los usuarios de una manera similar a la comunicación humana.

4. ¿A qué equivale una palabra del promt?

Una "palabra en el prompt" se refiere a una palabra específica que se proporciona como parte de una consulta.
Esta palabra es esencial para determinar el contexto y la dirección de la conversación. 
Cada palabra en el prompt es fundamental para que el modelo comprenda tu solicitud y genere una respuesta.
Por lo tanto, una palabra en el prompt se refiere a una unidad básica de información dentro de la entrada que
el usuario proporciona, que influye en cómo se desarrolla la interacción.
